# Resources

Templates, examples, and community resources.

## Templates

### Full Stack Template

A complete full-stack template with SQLite, JWT authentication, and ML inference.

```bash
git clone git@github.com:ukiapi/full-stack-ukiapi-template.git
cd full-stack-ukiapi-template/backend
ukiapi dev
```

**Features:**
- SQLite database with SQLx
- JWT authentication
- CORS configuration
- Custom middleware
- ML inference with Tract ONNX
- Docker support

[Repository](https://github.com/ukiapi/full-stack-ukiapi-template)

### Basic Template

A minimal API template for getting started quickly.

```bash
ukiapi new my-api
cd my-api
ukiapi dev
```

## Examples

### Official Example

The `example/` directory in the main repository demonstrates all features:

```bash
git clone git@github.com:ukiapi/ukiapi.git
cd ukiapi/example
cargo run
```

**Includes:**
- CRUD operations
- Authentication
- WebSocket
- File uploads
- Background tasks
- Custom middleware
- Static files
